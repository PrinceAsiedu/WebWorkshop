CSS Selectors

Attribute Selectors

Attribute selectors let you select elements based on the presence of an attribute or
based on the value of an attribute. The syntax is square brackets, [], with the suitable
attribute inside. There are several variations that you can use to make matches:

  • [attribute] will select all elements with an attribute present; example, [href]
  • [attribute=value] will select all elements with an attribute with an exact value;example, [lang="en"]
  • [attribute^=value] will select all elements with an attribute with a value that begins with the matching value; for example, [href^="https://"]
  • [attribute$=value] will select elements with an attribute with a value that ends with the matching value; for example, [href$=".com"]
  • [attribute*=valule] will select elements with an attribute with a value that has a match somewhere in the string; for example, [href*="co.uk"]

Pseudo-classes

To select an element when it is in a particular state, we have several pseudo-classes
defined. The syntax of a pseudo-class is a colon, :, followed by a keyword.

  • When an anchor element has an href attribute, it will have the :link pseudo class applied to it
  • When a user hovers over the link, the :hover pseudo-class is applied to it.
  • When the link has been visited, it has the :visited pseudo-class applied to it.
  • When the link is being clicked, it has the :active pseudo-class applied to it
  • :first-child
  • :last-child
  • :nth-child
  • :nth-last-child
  • :first-of-type
  • :last-of-type
  • :nth-of-type
  • :nth-last-of-type.

Pseudo-elements

Pseudo-element selectors are preceded by two colons (::) and they are used to select part of an element. The available pseudo-elements include

  • ::after
  • ::before
  • ::firstletter
  • ::first-line
  • ::selection
  • ::backdrop.
